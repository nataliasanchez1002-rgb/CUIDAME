Tech Stack

* Python
* Kivy / KivyMD
* OpenCV
* TensorFlow Lite
* JSON for data storage
* Random, datetime, OS libraries

---
Data Storage

* `usuarios.json` → user login data
* `usuario\_deteccion\_riesgo.json` → chat history + risk level

---

Goal

To create an **interactive educational and safety-focused app** that combines learning, emotion recognition, and simple games in one platform.
